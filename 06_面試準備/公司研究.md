# 🏢 公司研究

> 面試前必須了解目標公司！

---

## 📌 NVIDIA

### 公司概覽

```
成立：1993 年
總部：Santa Clara, California
創辦人：Jensen Huang（黃仁勳）
市值：超過 1 兆美元（2024）

核心業務：
- GPU（繪圖處理器）
- AI/ML 加速器
- 資料中心解決方案
- 自動駕駛
- 遊戲

口號："The New AI Computing Company"
```

### 與 BMC 相關的產品

```
DGX 系列：
- DGX A100、DGX H100
- 企業 AI 訓練系統
- 需要強大的 BMC 管理

HGX 平台：
- GPU 基板平台
- 給 OEM 使用
- 需要 BMC 整合

NVIDIA BlueField DPU：
- 資料處理器
- 也有 BMC 管理需求
```

### NVIDIA 在 OpenBMC 的貢獻

```
NVIDIA 是 OpenBMC 的重要貢獻者：
- 開源很多 GPIO、I2C 相關的改進
- 積極參與 phosphor 專案
- 推動 PLDM/MCTP 支援
```

### 台灣辦公室

```
據點：台北、新竹
規模：數百人
職缺類型：
- BMC 韌體工程師
- System Software Engineer
- CUDA/Driver 工程師
```

### 面試流程

```
典型流程：
1. Recruiter 電話（30 分鐘）
2. 技術電話面試（1 小時）
3. 現場/視訊面試（4-5 輪）
   - 2-3 輪技術
   - 1-2 輪 BQ/文化
4. Hiring Committee 審核
5. Offer

準備重點：
- 技術深度（C、Linux、嵌入式）
- 專案經驗（能詳細說明）
- 文化適應（NVIDIA 的價值觀）
```

### NVIDIA 文化和價值觀

```
核心價值：
- Innovation（創新）
- Intellectual Honesty（知識誠實）
- Speed & Agility（速度和敏捷）
- One Team（團隊合作）

面試可能問：
- 你如何持續學習新技術？
- 給一個你快速解決問題的例子？
```

## 📌 AMD

### 公司概覽

```
成立：1969 年
總部：Santa Clara, California
CEO：Lisa Su（蘇姿丰）

核心業務：
- CPU（Ryzen、EPYC）
- GPU（Radeon）
- 資料中心解決方案
- 嵌入式處理器

近年發展：
- 2022 年收購 Xilinx（FPGA）
- EPYC 在伺服器市場佔有率提升
```

### 與 BMC 相關的產品

```
EPYC 伺服器平台：
- 企業級伺服器
- 需要 BMC 管理

Instinct 系列：
- MI250X、MI300 等
- AI/HPC 加速器
- 競爭 NVIDIA
```

### AMD BMC 特色

```
強調安全：
- Secure Boot
- SPDM 支援
- Root of Trust (RoT)
- Platform Security Processor (PSP)

OpenBMC 貢獻：
- 安全相關功能
- 特定平台支援
```

### 台灣辦公室

```
據點：台北、新竹
職缺類型：
- BMC Firmware Engineer
- Server Platform Engineer
- Embedded Software Engineer
```

### 面試流程

```
典型流程：
1. HR 初面
2. 技術電話面試（1-2 輪）
3. 現場/視訊面試（3-4 輪）
4. Offer

準備重點：
- OpenBMC 經驗
- 安全知識（Secure Boot）
- Linux kernel 基礎
```

## 📌 Intel

### 公司概覽

```
成立：1968 年
總部：Santa Clara, California

核心業務：
- CPU（Xeon、Core）
- FPGA（Altera）
- 資料中心解決方案
- 網路/儲存

規模：全球約 12 萬員工
```

### 與 BMC 相關

```
Intel Server Platform：
- Xeon Scalable 處理器平台
- 龐大的伺服器市場

OpenBMC：
- Intel 也是 OpenBMC 貢獻者
```

### 台灣辦公室

```
據點：台北
規模：較大的辦公室
職缺類型：
- BMC Software Engineer
- System Software Engineer
- Validation Engineer
```

### 面試流程

```
典型流程：
1. HR 篩選
2. 技術電話面試
3. 現場面試（多輪）
4. Offer

特色：
- 流程較完整
- 可能有 coding test
```

## 📌 其他值得考慮的公司

### 台灣 IC 設計公司

```
聯發科（MediaTek）：
- 有 server chip 發展
- 韌體職缺不少
- 薪資優渥

瑞昱（Realtek）：
- 網通晶片
- BMC 相關較少

創意電子、世芯電子：
- ASIC 設計服務公司
- 可能有韌體需求
```

### 伺服器 OEM

```
Supermicro：
- 總部在 San Jose
- 使用 OpenBMC
- 有台灣辦公室

Quanta（廣達）/ Inventec（英業達）：
- 大型 ODM
- 自有或使用 OpenBMC
```
