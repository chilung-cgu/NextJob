## 🔑 關鍵提醒

> **你之前的準備方向（LeetCode 演算法）對 IC 韌體面試幫助有限！**

IC 韌體面試考的是：
- ⭐ C 語言底層細節（指標、位元運算、`volatile`/`static`/`const`）
- ⭐ 硬體通訊協定（I2C、SPI、UART）
- ⭐ 中斷處理、記憶體管理
- ⭐ Linux 驅動程式基礎（如果涉及 Linux）
- ⭐ 你的專案經驗（細節會被深挖）

**不太考的**：
- ❌ 複雜演算法（DP、圖論）
- ❌ 系統設計（分散式系統那種）
